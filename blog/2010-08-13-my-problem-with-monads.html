My Problem with Monads


From time to time I will read something about monads, but so far I have utterly failed to understand them. In classic internet fashion, I am going to blame everyone else for my lack of comprehension. So here's my problem with monads: there seem to be plenty of tutorials out there that are able to define a monad as "a thing with a bind function and a return function and blah blah blah state blah blah blah representation of computation blah blah blah", but there seem to be few to none capable of answering the all-important follow-up question: "So what?".
